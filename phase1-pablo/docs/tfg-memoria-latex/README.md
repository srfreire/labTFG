# Memoria TFG en LaTeX

Carpeta creada a partir del modelo `GrEI_TFG_Modelo_B_Memoria.zip` localizado
en `~/Downloads`.

## Archivos principales

- `traballo.tex`: memoria principal.
- `caratula.tex`: carátula auxiliar.
- `presentacion.tex`: presentación Beamer inicial.
- `capitulos/`: capítulos traducidos y adaptados al sistema real de la fase 1.
- `figuras/`: logos y figuras originales del modelo.

## Compilación

Desde esta carpeta:

```bash
pdflatex traballo.tex
pdflatex traballo.tex
```

La segunda ejecución actualiza índice, referencias y bibliografía.
