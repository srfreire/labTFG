\chapter{Galería de la interfaz}
\label{ap:ui}

Este apéndice reúne una selección de capturas de la interfaz web del
laboratorio: la vista principal, el avance del pipeline, el \textit{replay} de
simulación, las gráficas comparativas, las trazas de decisión, el informe final
y la inspección del Knowledge Backbone.

Las figuras no se incluyen solo como muestra visual: documentan los estados
principales que el usuario debe poder inspeccionar durante una sesión completa.
