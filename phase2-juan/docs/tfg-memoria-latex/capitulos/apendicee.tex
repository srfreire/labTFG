\section{Salida de detalle de cada agente}

Además de la vista general, cada agente del pipeline produce una salida propia
en la conversación. El \textbf{Architect} traduce la descripción en lenguaje
natural a una especificación de entorno (Figura~\ref{fig:architect}); el
\textbf{Tracker} registra las trayectorias de cada agente
(Figura~\ref{fig:tracker}); el \textbf{Analyst} detecta patrones y compara los
modelos (Figura~\ref{fig:patrones}) y desglosa las trazas de decisión
paso a paso (Figura~\ref{fig:traces}); el panel lateral resume el estado del
pipeline (Figura~\ref{fig:pipeline}) y el \textbf{Reporter} compila el informe
PDF final (Figura~\ref{fig:reporter}). Por último, la interfaz expone el
\textit{Knowledge Graph} científico que los agentes consultan
(Figura~\ref{fig:kg}).
